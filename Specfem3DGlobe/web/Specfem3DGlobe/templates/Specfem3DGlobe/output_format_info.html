OUTPUT FORMAT

RECORD LENGTH IN MINUTES: The desired record length of the synthetic seismograms (in minutes). This controls the length of the numerical simulation, i.e., twice the record length requires twice as much CPU time.

SAVE MESH FILES: Turn this option on to save AVS (www.avs.com ), OpenDX (www.opendx.org ), or ParaView (www.paraview.org ) mesh files for subsequent viewing.

RECEIVERS CAN BE BURIED: Turn this option on to accommodate stations with instruments that are buried, i.e., the solver will calculate seismograms at the burial depth specifed in the STATIONS file.

PRINT SOURCE TIME FUNCTION: Turn this option on to print information about the source time function in the file OUTPUT FILES/plot_source_time function.txt.
