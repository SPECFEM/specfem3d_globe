1 # Specfem3D Globe
2 
3 Program specfem3D from `SPECFEM3D_GLOBE` is a 3-D spectral-element solver for the Earth.
4 It uses a mesh generated by program meshfem3D.
5 
6 Main "historical" authors: Dimitri Komatitsch and Jeroen Tromp
7  (there are currently many more!)
